# Simple Gallery App
This is app which display photo from [unplash](https://unsplash.com/developers). You can watch the [demo](https://githi54.github.io/gallery__app/).
<details>
  <summary>Get started</summary>
  Click to 'Load Photo' button 
  <br /> <br />
</details>

# Stack
1. Typescript
2. React, React Hooks
3. Bootstrap
4. axios
